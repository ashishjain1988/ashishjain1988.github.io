SPINNER Result Summary

  • UNIPROT ID – Uniprot Id of the protein in the network
  • GENE NAME – Gene Name of the particular protein (NA if gene name is not found)
  • DEGREE – The number of protein nodes connected with the protein in the protein-protein interaction network
  • INITIAL SCORE – Initial ranking score of the protein by using the following heuristic scoring function
  • INITIAL RANK – Initial Rank of the protein in the whole network on the basis of Initial score.
  • INITIAL RANK IN SUB-NETWORK - Initial Rank of the protein in the sub-network on the basis of Initial score.
  • ITERATIVE SCORE – Iterative Score of the protein by using the page rank algorithm and the initial score of the protein.
  • ITERATIVE RANK - Initial Rank of the protein in the whole network on the basis of Initial score.
  • ITERATIVE RANK IN SUB-NETWORK - Initial Rank of the protein in the sub-network on the basis of Initial score.
  • SEED FLAG – The flag(Y/N) whether the protein is present in the initial seed set or not.
  • SUB-NETWORK ID – The sub-network in which the protein is connected.
  • DELTA RANK – The difference between the initial and the iterative ranks of the protein.
    Delta rank=Initial Rank-Iterative Rank
  • DELTA – The delta score of the proteins using the initial and iterative rank.
    Delta=(Initial Rank-Iterative Rank)/(Initial Rank)
  • GINITIAL RANK – The global initial rank of the protein in the HAPPI Database
  • GINITIAL RANK DELTA – The delta rank score of proteins using the global and local rank.
    GINITIAL RANK DELTA=(Global Initial Rank-Local Initial Rank)/(Global Initial Rank)
  • GITERATIVE RANK - The global initial rank of the protein in the HAPPI Database
  • GITERATIVE RANK DELTA - The delta rank score of proteins using the global and local rank
    GITERATIVE RANK DELTA =(Global Iterative Rank-Local Iterative Rank)/(Global Iterative Rank)
